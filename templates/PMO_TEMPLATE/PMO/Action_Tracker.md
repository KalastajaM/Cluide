## Status key

| Status | Meaning |
|---|---|
| 🔴 Open | Action is open; no progress yet |
| 🔄 In progress | Action is actively being worked on |
| ⏳ Blocked / Awaiting | Action is blocked or awaiting input from another party |
| 🟡 Monitoring | Action is under observation; no immediate step required |
| 💤 Snoozed | Action is deprioritized; revisit later |
| ✅ Done | Action is complete |
